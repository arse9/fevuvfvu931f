About Us

At the Morse Code Translator, formerly Morse Decoder (Turkish Patent and Trademark Office: 2021/186923), we offer a free and easy-to-use Morse code converter that can be used to translate Morse code to text and text to Morse code.

In addition to Morse code translation, we offer an interactive page to help you learn Morse code, featuring both letters and numbers from the Morse code alphabet. We also provide detailed pages on the NATO phonetic alphabet and the significance of the SOS distress signal.

Lastly, our binary to Morse translator and Morse to binary translator allows easy conversion between Morse code and binary code.

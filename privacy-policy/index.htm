1. The Type of Personal Information We Collect

We currently collect and process only cookies, log files, IP addresses, timestamps, and user agents.

Any text you input to generate Morse code, or any input for translation from Morse code, is not saved. All processing is done directly in your browser, ensuring that only you can see it.

Similarly, any audio you generate on this site is made by your browser using our code. We don't save or post it anywhere. But remember, others can create the same sound if they use the same settings.
